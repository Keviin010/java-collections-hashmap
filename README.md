# Java Collections — HashMap Examples

## Descripción
Colección de ejemplos en Java centrados en el uso de `HashMap` para modelar relaciones entre objetos. Incluye implementaciones con clases modelo (`Persona`, `Direccion`) y demostraciones de operaciones básicas: inserción, acceso seguro, eliminación y recorrido.

Este repositorio está pensado como material práctico para reforzar conceptos de colecciones en contextos de backend.
