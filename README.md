# Java Collections — HashMap Examples

Repositorio en Java enfocado en el uso de colecciones, principalmente `HashMap`,.  para modelar relaciones entre objetos y simular operaciones comunes en contextos backend.

## Objetivo del proyecto
Reforzar el manejo de colecciones en Java mediante ejemplos prácticos que representen relaciones entre entidades del dominio, aplicando operaciones de inserción, acceso seguro, eliminación y recorrido de datos.

## Estructura del proyecto
El proyecto está organizado por paquetes según el tipo de colección utilizada:

- map: ejemplos con HashMap
- list: uso de listas
- set: uso de conjuntos
- modelo: clases de dominio como Persona y Direccion

## Conceptos aplicados
- Java Collections Framework
- HashMap
- List y Set
- Modelado de relaciones entre objetos
- Operaciones CRUD sobre colecciones
- Separación lógica por paquetes

## Funcionalidades
- Inserción y almacenamiento de objetos en HashMap
- Acceso seguro a los datos
- Eliminación de elementos
- Recorrido de colecciones
- Asociación entre entidades del dominio

## Tecnologías utilizadas
- Java
- Java Collections Framework

## Contexto
Este proyecto forma parte del proceso de aprendizaje de Java backend y está orientado a fortalecer el manejo de estructuras de datos fundamentales antes de integrarlas en aplicaciones backend más complejas.

## Próximo paso recomendado
- Integrar estas estructuras dentro de un proyecto backend más amplio
- Utilizar estas colecciones como base para repositorios y servicios.
